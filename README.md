## Installation Instructions

1. clone the repository

```powershell
git clone https://github.com/DailyTracks/client
```

2. initial configuration settings in nodejs

```powershell
npm install
```

3. run App.jsx

```powershell
npm start
```

or

```powershell
nodemon App.jsx
```
